🔒 Установка SSL сертификата на Windows 11

Для доступа к https://192.168.0.107:4321/ без ошибок безопасности, установите корневой сертификат.

Важно: После установки перезапустите браузер, если он был открыт.

⚡ Способ 1: PowerShell (Автоматический)

Самый простой способ — запустить PowerShell скрипт от имени администратора:

# 1. Скачайте скрипт
Invoke-WebRequest -Uri "http://192.168.0.107:8080/install-ca-windows.ps1" -OutFile "$env:TEMP\install-ca.ps1"

# 2. Запустите от имени администратора
Start-Process powershell -Verb runAs -ArgumentList "-ExecutionPolicy Bypass -File `"$env:TEMP\install-ca.ps1`""

Или просто скачайте и запустите:

Скачать PowerShell скрипт

🔧 Способ 2: Ручная установка

  1. Скачать сертификат
  2. Дважды кликните по скачанному файлу mkcert-ca.crt
  3. Нажмите "Установить сертификат"
  4. Выберите "Локальный компьютер" → "Далее"
  5. Выберите "Поместить все сертификаты в следующее хранилище"
  6. Нажмите "Обзор" и выберите "Доверенные корневые центры сертификации"
  7. Нажмите "ОК" → "Далее" → "Готово"

🌐 После установки

Откройте в браузере:

Открыть https://192.168.0.107:4321/

Браузер больше не должен показывать предупреждение о небезопасном соединении.

Если у вас возникли проблемы, убедитесь что:

  • Сертификат установлен для всех пользователей (Local Machine)
  • Браузер полностью закрыт и открыт заново
  • Вы используете URL с IP-адресом, указанным в сертификате (192.168.0.107)
